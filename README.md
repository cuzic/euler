# euler

cuzic's Project Euler solutions written in Clojure

## Usage 1 (in repl)

> lein repl

(in lein repl)

> (use 'euler/pe018)

> (pe018)

## Usage 2 (in command line)
> lein run pe018 # just runs problem 018
